# -Поддерживаемые языки:

В начале определен словарь languages, где указаны языковые коды и их названия (например, 'ru' — русский, 'en' — английский).
Функция выбора языка:

select_language(prompt) выводит пользователю список поддерживаемых языков и просит ввести код языка.
В цикле продолжается запрос, пока не будет введён правильный код.
Основной цикл:

Программа работает в бесконечном цикле, пока пользователь не решит выйти.
Запись голоса:

Предлагается говорить сейчас ("🎙️ Говори сейчас...").
В течение 5 секунд записывается голос пользователя через микрофон, с частотой 44,1 кГц.
Записанный аудиофайл сохраняется в output.wav.
Распознавание языка для транскрипции:

Пользователь выбирает язык для распознавания речи (например, русский или английский).
Распознавание речи:

Загружает записанный файл (output.wav) и использует API Google распознавания речи (speech_recognition) для преобразования аудио в текст.
Если распознавание удачно — выводит распознанный текст.
В случае ошибок — сообщает пользователю, что речь не распознана или была ошибка сервиса.
Выбор языка для перевода:

Пользователь выбирает язык, на который нужно перевести распознанный текст.
Перевод:

Используется модуль googletrans для перевода текста.
Переведённый текст отображается вместе с названием языка.
Вопрос о повторении:

Спрашивается, хотите ли продолжить переводить.
Если пользователь ввёл 'да', цикл повторяется.
Иначе программа завершается с прощанием.
